# Ops Standard Local

Bản web local chạy trên Ubuntu hiện có, không cần cài thêm package hệ thống.

## Tính năng
- Đăng nhập
- Phân quyền: admin / operator / viewer
- Backup dữ liệu nội bộ
- Xuất file JSON / CSV

## Chạy
```bash
cd /root/.openclaw/workspace/ops-standard
npm start
```

Mặc định chạy ở: `http://127.0.0.1:3080`

## Tài khoản mặc định
- user: `admin`
- pass: `admin123!`

**Đổi mật khẩu ngay sau khi vào vận hành thực tế.**
