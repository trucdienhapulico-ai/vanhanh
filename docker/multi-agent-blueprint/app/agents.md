# Agent Catalog v1

## Chief Coordinator

### tocodienbot
- Role: tổng hợp, điều phối, theo dõi, báo cáo
- Not owner of deep specialty work
- Escalates to human when budget, shutdown, or safety thresholds are crossed

## Manager Agents

### Ops Manager
- Quản lý vận hành ca
- Theo dõi trạng thái hệ thống
- Điều phối checklist đầu ca/cuối ca

### Maintenance Manager
- Quản lý PM/CM
- Theo dõi backlog, lịch bảo trì, phụ tùng

### Incident Manager
- Điều phối xử lý sự cố
- Phân severity, tạo bridge xử lý, đẩy RCA sơ bộ

### Projects Manager
- Quản lý nâng cấp, triển khai, lắp đặt

### Safety & Compliance Manager
- Kiểm soát SOP, LOTO, an toàn điện, tuân thủ

### Inventory & Procurement Manager
- Quản lý vật tư, tồn kho, đề xuất mua

## Specialist Agents

### Electrical Specialist
- Tủ điện, MCCB/MCB, ATS, nguồn lực, mất pha, quá tải

### Pump & Irrigation Specialist
- Bơm, tưới, áp lực, valve, schedule, line issues

### PLC & Automation Specialist
- PLC, relay logic, I/O, HMI, interlock, alarm flows

### HVAC Specialist
- Điều hòa, thông gió, AHU/FCU, thermostat, control logic

### Lighting Specialist
- Chiếu sáng sân, đường, khu dịch vụ, timer/contactors

### Generator Specialist
- Máy phát, ATS, chuyển nguồn, nhiên liệu, test run

### Water Treatment Specialist
- Chất lượng nước, bơm hóa chất, lọc, tuần hoàn

### Documentation Specialist
- SOP, checklist, form mẫu, hướng dẫn thao tác

### Reporting Analyst
- KPI, downtime, MTTR, MTBF, thống kê sự cố

## Governance Agents

### QA Critic Agent
- Rà logic đầu ra
- Bắt thiếu bước, thiếu dữ liệu, mâu thuẫn nội bộ

### Risk Agent
- Chấm rủi ro an toàn/vận hành/dịch vụ

### Audit Trail Agent
- Lưu decision log, owner, timestamp, rationale

### Knowledge Agent
- Biến bài học thành SOP/runbook/FAQ
