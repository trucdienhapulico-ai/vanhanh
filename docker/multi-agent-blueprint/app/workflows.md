# Workflow Definitions v1

## 1. Incident Workflow
- Intake
- Severity classification
- Ownership assignment
- Specialist investigation
- Mitigation proposal
- QA/Risk review
- Human approval if critical
- Resolution summary
- RCA capture

## 2. Preventive Maintenance Workflow
- Pull maintenance schedule
- Check due tasks
- Verify manpower/tools/spares
- Execute checklist
- Record anomalies
- Escalate findings
- Update backlog and next due date

## 3. Project Workflow
- Intake objective
- Site/system assessment
- Scope definition
- Risk review
- Resource estimate
- Approval gate
- Execution tracking
- Commissioning & handover

## 4. Reporting Workflow
- Collect incident/maintenance/project data
- Normalize records
- Calculate KPIs
- Highlight red flags
- Produce concise management summary
